# Rovio - Real-time visual inertial odometry
## Demo
https://youtu.be/maiFTbCU-PY

## Hardware Setup
### Components
- Computer: Raspberry Pi 4 B+
- IMU: MPU 6050
- Camera: Pi Camera V2 - 8 MP
- Battery: Redmi PB100LZM Powerbank (5V, 1k mAh)
- Structural: Shoe box :)
